Turtle Graphics
Python Tutorial

Start by Importing the turtle Module
Whith the imort turtle Command

Move the turtle forward
With the turtle.forward([pixels]) command

Move the turtle backward
With the turtle.backward([pixels]) command

Turn left or right by [x] degrees
With the turtle.{left/right}(x) command

turtle.penup() Restricts the turtle from drawing when moving

draw a circle with the turtle.circle([radius],[degrees]) command

A shortcut to starting cords and direction is the turtle.home() command

Change the Turtle's pencolor with the turtle.color("[color]")

Fill a shosen shape with the turtle.begin_fill(){initially} & turtle.end_fill(){finally} commands

Change the full background canvas color with the turtle.bgcolor("[color]")
